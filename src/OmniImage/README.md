### 実行&使い方
実行時に引数として読み込むファイルやウィンドウサイズを指定します
```bash
#ウィンドウサイズを指定しない場合
./OmniImage filename

#ウィンドウサイズを指定する場合
./OmniImage filename window_width window_height
```
回転操作などはキーボード操作で行います.操作方法は以下の通りです

```bash
#回転操作
A , D     :　左右方向に視線を移動する

W , S     :　上下方向に視線を移動する

E , Q     :　時計回り、反時計回りに画面を回転させる

R         :  回転のリセット

#それ以外
Escape    :　アプリ終了

"+" , "-" :  ズームイン/ズームアウト
```
